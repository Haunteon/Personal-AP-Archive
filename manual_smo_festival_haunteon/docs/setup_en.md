# Manual Randomizer Setup Guide

## Required Software

- Super Mario Odyssey for the Switch
- Archipelago Launcher

## Installation Procedures

- Place the apworld in the custom_worlds folder
- Relaunch Archipelago Launcher and create a yaml (Options Creator or Generate Template)

## Joining a MultiWorld Game

Host
- place yaml in players folder
- generate multiworld
- host on website or through archipelago launcher
- post room link

Non-Host
- give yaml to host
- open manual client
- connect to room
- connect to slot
